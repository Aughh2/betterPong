# betterPong

Experimental Pong-inspired game built in Godot.  
Focuses on **component-based entity management** and flexible architecture.

- Lightweight custom ECS for entity management
- Scalable design for adding arenas, paddles, and balls
- Explores **modularity, decoupling, and code organization**
- Work in progress; primarily highlights architectural decisions
